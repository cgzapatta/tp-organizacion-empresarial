
# Trabajo Práctico - Gestión Colaborativa con Git, GitHub y Jira

## Integrante
- German Zapatta

## Escenario seleccionado
Escenario B - Análisis de Ventas de una Pequeña Empresa

## Descripción del proyecto
Este proyecto realiza un análisis básico de ventas utilizando Python, Git, GitHub y Jira.

El análisis permite:
- calcular ventas totales
- identificar el producto más vendido
- generar un gráfico de ventas por producto

## Estructura del repositorio

- /datos → dataset utilizado
- /scripts → scripts Python
- /resultados → gráficos y resultados

## Dataset utilizado
Dataset simulado de ventas comerciales en formato CSV.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Git
- GitHub
- Jira
- Google Colab

## Instrucciones de ejecución

Ejecutar:

python scripts/analisis_ventas.py

El gráfico generado se almacena en:
resultados/grafico_ventas.png
